CS 184: Computer Graphics and Imaging, Spring 2021

Final Project Proposal: Music-reactive Real-time Fire Simulation

Henry Allen, Priyanka Kargupta, Hannah Ku, Amber Xie



Summary

In this project, we have implemented a program that allows us render SVG files through the rasterization of triangles, as well as performing PNG texture mapping on them. For triangle rasterization, we progressed from simple point-in-line tests to utilizing barycentric coordinates/interpolation. We also implement various pixel and level sampling methods for texture mapping. Overall, this project demonstrated that with relatively small adjustments and utilizing different algorithms, we can achieve much better results when rendering-- although the mistakes in rendering made along the way are at times more interesting!

We check every sample in the bounding box (determined based on the minimums and maximums of the vertex coordinates) and color it if necessary. We used the line equation, which we plugged in each point we examine to see if the equation returns a positive or negative value. If the point returns all positive values for all three lines or all negative values, then we color it. We consider both the case of all negative and all positive to cover both counterclockwise and clockwise orientation of the points.
